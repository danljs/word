Notice: This app can only run in IE browser.
Before run this app, you have to enable ActiveXObject in IE.
  • Go to Tools-->Internet Options
  • Select security tab
  • Click on Trusted Sites (or Local Intranet depending on whether your site is trusted or not)
  • Click on Custom Level
  • Ensure that "Initialize and script active x controls is not marked safe for scripting" is enabled - this comes under Activex controls and plug-ins section towards 1/4th of the scroll bar.
  • Click OK, OK.
